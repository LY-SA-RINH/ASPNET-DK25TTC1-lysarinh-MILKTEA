** 2. Giới thiệu đề tài

MilkTea House là website bán trà sữa được xây dựng bằng ASP.NET Core MVC và SQL Server.

Website hướng đến việc hỗ trợ khách hàng xem sản phẩm, xem thông tin sản phẩm, thêm sản phẩm vào giỏ hàng và đặt hàng trực tuyến.

Hệ thống dự kiến có các chức năng dành cho khách hàng, nhân viên và quản trị viên.

** 3. Công nghệ sử dụng

- Ngôn ngữ lập trình C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML
- CSS
- Bootstrap
- Visual Studio
- Git
- GitHub
- GitHub Desktop

** 4. Các chức năng đã thực hiện

*** 4.1. Chức năng phía khách hàng

- Khởi tạo dự án ASP.NET Core MVC.
- Xây dựng giao diện trang chủ theo tông màu be và nâu.
- Xây dựng phần đầu trang, thanh điều hướng, banner và khu vực sản phẩm nổi bật.
- Tạo hiệu ứng chuyển slide cho banner.
- Hiển thị sản phẩm từ cơ sở dữ liệu lên trang chủ.
- Xây dựng trang danh sách sản phẩm và phân trang.
- Xây dựng trang chi tiết sản phẩm.
- Tìm kiếm sản phẩm theo tên và mô tả.
- Xây dựng giỏ hàng bằng Session.
- Cập nhật số lượng và xóa sản phẩm khỏi giỏ hàng.
- Xây dựng chức năng thanh toán và đặt hàng.
- Lưu đơn hàng, chi tiết đơn hàng và trừ tồn kho.
- Xây dựng chức năng đăng ký, đăng nhập và đăng xuất.
- Xây dựng chức năng quên mật khẩu và đặt lại mật khẩu.
- Xây dựng chức năng thông tin cá nhân và đổi mật khẩu.
- Xây dựng chức năng Đơn hàng của tôi.
- Cho phép khách hàng xem chi tiết và hủy đơn khi còn ở trạng thái Chờ xác nhận.
- Xây dựng trang Liên hệ và lưu nội dung vào cơ sở dữ liệu.

*** 4.2. Chức năng phía nhân viên và quản trị viên

- Phân quyền Khách hàng, Nhân viên và Quản trị viên.
- Quản lý đơn hàng và cập nhật trạng thái đơn hàng.
- Hoàn lại tồn kho khi đơn hàng bị hủy.
- Quản lý sản phẩm, tìm kiếm, lọc trạng thái và phân trang.
- Thêm và sửa sản phẩm, tải hình ảnh sản phẩm.
- Mở bán hoặc tạm ngưng bán sản phẩm.
- Quản lý danh mục sản phẩm.
- Quản lý khách hàng và trạng thái tài khoản.
- Xây dựng trang Tổng quan quản trị.
- Xây dựng Báo cáo doanh thu.
- Xây dựng chức năng Quản lý liên hệ.
- Đánh dấu liên hệ Đã xử lý hoặc mở lại.
- Hiển thị số liên hệ chưa xử lý trên trang Tổng quan quản trị.

*** 4.3. Cơ sở dữ liệu và quản lý mã nguồn

- Tạo cơ sở dữ liệu MilkTeaHouse trên SQL Server.
- Kết nối website với SQL Server bằng Entity Framework Core.
- Tích hợp ASP.NET Core Identity.
- Tạo và cập nhật Migration trong quá trình phát triển.
- Quản lý mã nguồn bằng Git, GitHub và GitHub Desktop.
- Thực hiện commit và push sau các chức năng quan trọng.

** 5. Dữ liệu sản phẩm hiện tại

Cơ sở dữ liệu hiện có 24 sản phẩm mẫu thuộc các danh mục trà sữa và thức uống khác nhau.

Một số sản phẩm tiêu biểu:

- Trà sữa Matcha
- Trà sữa đường đen
- Trà sữa truyền thống
- Trà sữa Socola
- Trà sữa Oolong
- Trà đào cam sả
- Trà sữa Caramel

Dữ liệu sản phẩm được đọc từ cơ sở dữ liệu MilkTeaHouse và hiển thị trên trang chủ, trang danh sách sản phẩm và trang chi tiết sản phẩm.

** 7. Cấu trúc thư mục

- `.github`: chứa các tập tin cấu hình liên quan đến GitHub.
- `src`: chứa mã nguồn dự án ASP.NET Core MVC.
- `progress-report`: chứa báo cáo tiến độ thực hiện đồ án hàng tuần.
- `thesis/doc`: chứa báo cáo đồ án định dạng Word.
- `thesis/pdf`: chứa báo cáo đồ án định dạng PDF.
- `thesis/html`: chứa tài liệu dạng web nếu có.
- `thesis/abs`: chứa slide PowerPoint và tài liệu phục vụ báo cáo.
- `thesis/refs`: chứa tài liệu tham khảo.
- `.gitignore`: quy định các tập tin không đưa lên GitHub.
- `README.md`: giới thiệu và hướng dẫn sử dụng dự án.

** 9. Hướng dẫn chạy dự án

*** Bước 1: Cài đặt phần mềm

Cài đặt các phần mềm cần thiết:

- Visual Studio
- SQL Server Express
- SQL Server Management Studio
- GitHub Desktop

Khi cài Visual Studio, cần chọn thành phần:

`ASP.NET and web development`

*** Bước 2: Tải mã nguồn

Clone repository từ GitHub về máy tính bằng GitHub Desktop.

*** Bước 3: Mở dự án

Mở thư mục `src`, sau đó mở file Solution của dự án bằng Visual Studio.

*** Bước 4: Chuẩn bị cơ sở dữ liệu

- Mở SQL Server Management Studio.
- Kết nối đến máy chủ `.\SQLEXPRESS`.
- Kiểm tra cơ sở dữ liệu `MilkTeaHouse`.
- Đảm bảo cơ sở dữ liệu đã có dữ liệu sản phẩm mẫu.

*** Bước 5: Kiểm tra chuỗi kết nối

Kiểm tra chuỗi kết nối trong file `appsettings.json` để bảo đảm tên máy chủ và tên cơ sở dữ liệu phù hợp với máy đang sử dụng.

*** Bước 6: Chạy website

Mở dự án bằng Visual Studio và nhấn nút Run để chạy website.
